## 目的

- 基于 redis 的分布式锁

- 基于 oracle 的分布式锁

- 基于 mysql 的分布式锁

# 快速开始 

## 需要 

jdk1.7+

maven 3.x+

## maven 引入 

```xml
<dependency>
    <groupId>com.github.houbb</groupId>
    <artifactId>lock</artifactId>
    <version>${最新版本}</version>
</dependency>
```
